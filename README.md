# 🌐 GSB WiFi Automator

A specialized utility developed to automate the login process for university Wi-Fi captive portals, eliminating the need for manual browser-based authentication.

---

## 🚀 Features
- **One-Click Login:** Automates the login process by managing `curl` requests in the background.
- **Session Management:** Capable of starting and stopping network requests to keep the connection active.
- **User-Friendly Interface:** Built with a clean WPF UI for easy configuration and status tracking.

## 🛠 Tech Stack
- **Language:** C#
- **UI Framework:** WPF (.NET 4.8)
- **Networking:** Automated `curl` command handling.
